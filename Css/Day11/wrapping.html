Box Model

Box Model is the core model to handle the layout and design.They have 4 part is very important :

  1. content
  2. padding
  3. border
  4. margin

Padding

Content around space is the padding.If you have a content need to use padding

                Example:
                *{
                    Padding:0;
                    margin:0;
                    box-sizing:border:box;
                }
