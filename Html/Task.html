25)What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used for creating webpages and web applications. HTML provides the basic structure of a webpage by using different tags and elements. It tells the browser how content should be displayed on the screen.

HTML is considered the foundation of web development because every webpage on the internet uses HTML in some form. It helps organize and structure content such as headings, paragraphs, images, tables, links, forms, audio, and videos. HTML works together with CSS for styling and JavaScript for adding interactivity to websites.

26)HTML Tags

HTML tags are predefined instructions enclosed within angle brackets. Tags tell the browser how to display and organize webpage content. Tags are the basic building blocks of HTML and are used to create webpage structures.

Opening Tag

An opening tag marks the beginning of an HTML element. It informs the browser where the element starts. Opening tags are written inside angle brackets.

Example: <p>

The above tag indicates the starting point of a paragraph element.

Closing Tag

A closing tag marks the end of an HTML element. It contains a forward slash before the tag name. Closing tags tell the browser where the content of an element ends.

Example: </p>

Without closing tags, browsers may not correctly understand the webpage structure.

Self Closing Tag

Self-closing tags do not require separate closing tags because they complete their function within a single tag itself.

Examples: <br> <img> <hr>

These tags perform actions like inserting line breaks, displaying images and creating horizontal lines.

27)HTML Elements

HTML elements are created using an opening tag, content and a closing tag. Elements represent individual components displayed on webpages.

Example: <h1>Welcome</h1>

In this example, h1 is the tag and "Welcome" is the content displayed on the webpage.

Nested Elements

Nested elements are elements placed inside other elements. Nesting creates parent-child relationships and helps organize webpage content efficiently.

Example: <p> <b>Hello</b> </p>

In this example, the bold tag is placed inside the paragraph tag.

Root Element

The html tag is known as the root element because it contains all other elements in an HTML document. Every webpage starts and ends with the html tag.

Container Elements

Container elements are tags that contain opening and closing tags with content inside them. These elements hold text or other HTML elements.

Example: <p>Content</p>

Child Elements

Elements placed inside parent elements are called child elements. Child elements help create webpage hierarchy and structure.

Example: If a paragraph tag exists inside a div tag, then the paragraph becomes the child element.

28)Block Level Elements

Block-level elements are HTML elements that occupy the full available width of a webpage and always begin on a new line. These elements create separate sections and automatically push the next content onto a new line. Block elements are commonly used for structuring and organizing webpage layouts.

Examples of block-level elements include:

  • <div>
  • <p>
  • <h1> to <h6>
  • <table>
  • <section>

For example, a paragraph tag always starts on a new line and occupies the available width.

29)Inline Elements

Inline elements only occupy the space required by their content and do not start on a new line. These elements appear within the flow of text and are commonly used to format small portions of content.

Examples of inline elements include:

  • <span>
  • <b>
  • <i>
  • <a>
  • <img>

Unlike block-level elements, multiple inline elements can appear on the same line.

30)Semantic Tags

Semantic tags are HTML tags that clearly describe the meaning and purpose of the content enclosed within them. These tags improve readability, accessibility and search engine optimization. They help developers and browsers understand webpage structure more effectively.

Examples of semantic tags:

  • <header>
  • <footer>
  • <section>
  • <article>
  • <nav>

Semantic tags make HTML code more meaningful and easier to maintain.

31)Non Semantic Tags

Non-semantic tags do not provide any information about the meaning of content. They simply act as containers for styling or grouping elements.

Examples:

  • <div>
  • <span>

Developers usually combine these tags with CSS and JavaScript for webpage design and functionality.

32)Attributes

Attributes provide additional information about HTML elements and modify their behavior or appearance. Attributes are written inside the opening tag and usually contain a name and value pair.

Example: 

<img src="image.jpg" width="200">

Here, src and width are attributes.

Global Attributes

Global attributes are attributes that can be used with almost all HTML elements. They provide common properties that help identify or style elements.

Examples:

  • id
  • class
  • style
  • title

These attributes are useful for applying CSS and JavaScript functionality.

Local Attributes

Local attributes are specific to particular HTML elements and cannot be used everywhere.

Examples:

  • src → used with image tags
  • href → used with anchor tags
  • alt → used with images

These attributes provide special functions for specific elements.

33)Formatting Tags

Formatting tags are used to change the appearance and presentation of text content. They improve readability and help highlight important information.

Examples:

  • <b> → Bold text
  • <i> → Italic text
  • <u> → Underline text
  • <mark> → Highlight text
  • <strong> → Important text

33)Interactive Tags

Interactive tags allow users to interact with webpage content. These tags help create clickable elements and user actions.

Examples:

  • <a> Link tag
  • <button> Button tag
  • <details> Expandable section

Interactive elements improve user experience and website functionality.

34)Data Collection

Data collection is the process of gathering information from users through webpages. Websites collect information for registration, feedback, login and surveys.

HTML forms are commonly used for collecting user information.

35)Form Elements

Form elements are used to create forms and collect user data. Forms allow users to submit information to websites.

Examples:

  • <form>

  • <textarea>
  • <button>
  • <label>

Input Attributes

Input attributes provide additional settings and controls for input fields.

Examples:

  • type
  • placeholder
  • required
  • maxlength
  • value

These attributes help validate and customize user input.

Select

The select element creates a dropdown list allowing users to choose options from predefined values.

It is commonly used in registration forms and surveys.

Field Set

The fieldset tag groups related form elements together inside a form. It creates a border around grouped content and improves organization.

Fieldsets improve form readability.

36)Table Elements

37)Media Tags

Media tags are used to include multimedia content such as audio and videos in webpages.

Examples:

  • <audio>
  • <video>

38)Iframe

Iframe stands for Inline Frame. It allows another webpage, map or external content to be embedded inside a webpage.

Iframes are commonly used for embedding videos and maps.

39)Meta Tag

Meta tags provide information about webpages such as description, keywords, character set and author information.

Meta tags are placed inside the head section and are important for search engine optimization.

40)Fav Icon
