
Questions:

1) What is HTML and why is it called a markup language?
HTML (HyperText Markup Language) is the standard language used to create web pages. It is called a markup language because it uses tags to structure and describe the content of a webpage, such as headings, paragraphs, images, and links.

2)What is the difference between HTML and HTML5?
HTML is the older version used for creating webpages.
HTML5 is the latest version with new features like audio, video, semantic tags (<header>, <section>), canvas, and better support for multimedia and APIs.

3)What is the purpose of <!DOCTYPE html> in HTML?
<!DOCTYPE html> tells the browser that the document uses HTML5. It helps the browser display the webpage correctly and follow standard rules.

4)What is the difference between tags and elements in HTML?
Tag: The markup symbols written inside angle brackets, like <p> or <h1>.
Element: Includes the opening tag, content, and closing tag. Example: <p>Hello</p>.

5)What are semantic tags? Give examples.
Semantic tags clearly describe the meaning of the content inside them. They improve readability and SEO.
Examples: <header>, <footer>, <article>, <section>, <nav>.

6)What is the difference between semantic and non-semantic tags?
Semantic tags: Describe the purpose of content. Example: <article>, <nav>.
Non-semantic tags: Do not describe content meaning. Example: <div>, <span>.

7)What are meta tags and why are they used?
Meta tags provide information about the webpage, such as description, keywords, and character encoding. They help browsers and search engines understand the page.

8)Why do we use the <source> tag inside audio and video tags?
The <source> tag is used to specify multiple media files in different formats so browsers can choose a supported format.

9)What is iframe and where is it used in real-time applications?
An <iframe> is used to embed another webpage inside a webpage. Real-time uses include embedding maps, YouTube videos, advertisements, and external websites.

10)What is the difference between block-level elements and inline elements?
Block-level elements: Start on a new line and take full width. Examples: <div>, <p>, <h1>.
Inline elements: Do not start on a new line and take only required space. Examples: <span>, <a>, <b>.
